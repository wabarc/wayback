---
title: Wayback to Ghost Archive
---

Wayback relies on [Ghost Archive](https://ghostarchive.org/) as an upstream service for wayback and playback.
You can enable or disable this feature using the `--ga` flag or the `WAYBACK_ENABLE_GA` environment variable, which is enabled by default.

The code for wayback's implementation of the Ghost Archive integration can be found in the [wabarc/ghostarchive](https://github.com/wabarc/ghostarchive) repository.
